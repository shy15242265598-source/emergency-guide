# 应急智导 - AI应急科普小程序

一个用于应急安全知识科普的H5小程序，包含元力闯关、VR实训、AI安全顾问、短视频库等功能。

## 功能特点

- **元力闯关**：急救知识答题模块，包含地震、火灾、AED使用、心肺复苏四类关卡
- **VR实训**：虚拟实训模块，体验火灾、地震、急救等场景
- **AI安全顾问**：智能问答模块，解答急救相关问题
- **短视频库**：急救知识视频模块，包含AED使用、海姆立克急救法、心肺复苏术、创伤处理等视频

## 如何使用

### 方法一：本地打开
1. 下载或克隆本项目
2. 双击 `index.html` 文件即可在浏览器中打开

### 方法二：在线访问
